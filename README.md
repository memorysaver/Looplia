# Looplia
Looplia — an AI‑powered personal content agent that automates media creation and scheduling with human‑in‑the‑loop validation and long‑term memory feedback.

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **API**: TRPC for type-safe APIs
- **Routing**: TanStack Router
- **Build**: Turborepo monorepo setup
- **Package Manager**: pnpm

## Getting Started
```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```
